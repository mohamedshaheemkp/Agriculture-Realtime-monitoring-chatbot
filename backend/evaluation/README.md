# Evaluation

This directory contains tools and artifacts for evaluating the performance of the agricultural monitoring models.

## Directory Structure

- **scripts/**: Contains scripts for running model evaluations (e.g., calculating metrics, generating confusion matrices).
- **results/**: Stores the raw outputs from evaluation runs.
  - **val/**: Results from validation set evaluations.
  - **test/**: Results from test set evaluations.
  - **predictions/**: Saved model predictions for analysis.
- **reports/**: High-level summary reports (e.g., generated PDFs, Markdown summaries) derived from the results.
